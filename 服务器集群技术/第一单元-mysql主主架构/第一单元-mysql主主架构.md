# 第一单元-mysql主主架构

MySQL主主同步和主从同步的原理一样，只是双方都是主从角色。

什么叫主主复制？就是两个 MySQL 都能读能写，数据记录通过二进制传达给对方从而保持数据的一致性。

MySQL 主从复制官方文档：http://dev.mysql.com/doc/refman/5.6/en/replication.html
